My Portfolio

Master's Thesis - 3D Reconstruction of Complex Materials

A novel approach to 3D reconstruction, which leverages Helmholtz reciprocity to extract depth and normals of objects from 2D images which showed promising results.

Vision - Finding starfish in images using classic computer vision techniques

The code for this project includes a pipeline that takes an input image and outputs bounding boxes around starfish and the number of starfish found.

Vision - Visual search program using feature spaces

A visual search program that takes a query image and finds similar images from a database. To do this different feature spaces are created to sort images based on similarity.

Signal Processing - Reconstruction of a vowel sound

This program contains a pipeline to reconstruct a vowel sound by using formant and fundemental frequenices found from the original sound. This is commonally used in compression.

IOT - Varying ouput of sensors based on activity

A program in C that takes sensor readings from IOT devices and outputs aggregated values based on the current activity level. There are currently three levels. This was done in Contiki on Linux.

C# - A program to detect if a word is a palindrome

A program that takes an inputed sequence of characters and outputs if that sequence is a palindrome. A palindrome is a word that is spelt the same forwards and backwards e.g. "radar".
